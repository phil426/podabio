# Lefty Widget Gallery Thumbnails

This document specifies all widget thumbnails needed for the Lefty admin interface widget gallery.

## Image Specifications

- **Dimensions**: 480px × 240px (2:1 aspect ratio)
- **Format**: PNG or JPG
- **Display**: Cards are minimum 240px wide; images use `object-fit: cover` and will be cropped to fit
- **Style**: Clean, modern, minimalist design that clearly represents the widget type
- **Background**: Light neutral background (white or light gray) with subtle shadows/borders
- **Color Scheme**: Use PodaBio brand colors where appropriate, or neutral professional colors

## Design Guidelines

1. **Consistency**: All thumbnails should follow a consistent visual style
2. **Clarity**: Each thumbnail should immediately communicate what the widget does
3. **Branding**: Use PodaBio brand colors (#2563EB blue) where appropriate
4. **Simplicity**: Keep designs clean and uncluttered
5. **Recognition**: Icons and symbols should be universally recognizable
6. **Contrast**: Ensure good contrast for visibility on light backgrounds
